# Twitter Clone

Real-time CRUD, user authentication and file upload with serverless Firebase architecture.

-   **Language** : React.js, HTML/CSS/JavaScript
-   Firebase Authentication
-   Cloud Firestore
-   Email, Google, Github, Social Authentication
-   Tweet CRUD
-   File Upload
